utf8encode
==========

Encode binary files to valid UTF-8.


Installation
=======

Run `make`.


Usage
=======

Decoding and encoding a file will create files with .decoded/.encoded prefixes.

In addition to the created files, it will also be displayed via stdout in hex.


Encode a file
`./utf8encode -e <file>`

Decode a file
`./utf8encode -d <file>`
